# How to make clickable signs

## Step 1
Place a sign

![Place A Sign](/img/place-a-sign.png)

*After this, type whatever text you want on the sign and style it in any way you want*

## Step 2
Get the block property editor

![a](/img/get-the-block-property-editor.png)

## Step 3
Click sign with block property editor

![click-sign-with-block-property-editor.png](/img/click-sign-with-block-property-editor.png)

## Step 4

Type any command in the "On Interaction Command" input. Now when players click the sign the command runs as that player.
