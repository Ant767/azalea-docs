# Chat Rank Formats

| Tag             | Description                   |
| --------------- | ----------------------------- |
| #BC             | Bracket Color                 |
| #RC             | Default Rank Color            |
| #R(separator)   | Ranks                         |
| #NC             | Name Color                    |
| #MC             | Message Color                 |
| #S(objective)   | Score                         |
| #S(,objective)  | Score (with comma separators) |
| #M              | Message                       |
| #P              | Player Name                   |

# How To Change

Use `!chatrankformat change <format>`
