# Colors

# Changing name color

You can use admin panel to change name color, or alternatively you can do `/tag <player> add "name-color:color-code"`

# Changing bracket color

You can use admin panel to change bracket color, or alternatively you can do `/tag <player> add "bracket-color:color-code"`

# Changing message color

You can use admin panel to change message color, or alternatively you can do `/tag <player> add "messaage-color:color-code"`
